LITERAL SOUNDS LABS
macOS INSTALLATION INSTRUCTIONS

These instructions apply to Literal Sounds Labs plugin downloads for macOS.
Install only the formats included in your download.

BEFORE INSTALLING

1. Close your DAW and any standalone version of the plugin.
2. Double-click the downloaded ZIP file to extract it.
3. Do not rename the plugin files or remove files from inside a plugin bundle.

INSTALL THE PLUGIN

VST3

Copy the file ending in .vst3 to:

Macintosh HD/Library/Audio/Plug-Ins/VST3/

In Finder, choose Go > Go to Folder and enter:

/Library/Audio/Plug-Ins/VST3/

AU / AUDIO UNIT

Copy the file ending in .component to:

Macintosh HD/Library/Audio/Plug-Ins/Components/

In Finder, choose Go > Go to Folder and enter:

/Library/Audio/Plug-Ins/Components/

STANDALONE APPLICATION

If the download includes a file ending in .app, drag it to:

/Applications/

You only need to install the formats you use. For example, Ableton Live can use
VST3 and Audio Units, while Logic Pro uses Audio Units.

FIRST LAUNCH

1. Reopen your DAW.
2. Allow the DAW to scan newly installed plugins.
3. Create an audio track for effects or an instrument/MIDI track for synthesizers.
4. Load the plugin from the Literal Sounds Labs or manufacturer list.

Paid plugins can be opened in trial mode where available. To unlock a purchased
copy, enter the Payhip licence key in the plugin's licence section and select
Activate.

IF macOS BLOCKS THE PLUGIN OR APP

Only continue if the file was downloaded from the official Literal Sounds Labs
website.

1. Try to open the standalone app or load the plugin once.
2. Open System Settings > Privacy & Security.

On older macOS versions, open:

System Preferences > Security & Privacy > General

3. Look for a message about the blocked Literal Sounds Labs software.
4. Select Open Anyway or Allow Anyway, then try again.
5. Restart the DAW after approving the software.

If a standalone app is blocked, you can also Control-click the app, choose Open,
and confirm.

IF THE PLUGIN DOES NOT APPEAR

1. Confirm that the plugin was copied to the correct system folder.
2. Confirm that your DAW supports the installed format.
3. Open your DAW's plug-in manager and run a full rescan.
4. For Audio Units in Logic Pro, open Plug-in Manager and use Reset & Rescan
   Selection.
5. Restart the Mac if the plugin still does not appear.
6. Make sure you have not placed the plugin inside an extra nested folder.

UPDATING

1. Close the DAW.
2. Replace the old .vst3, .component or .app with the newer version.
3. Reopen the DAW and rescan plugins if required.

Your presets, settings, trial information and licence data are stored separately
from the plugin file, so replacing the plugin normally does not remove them.

UNINSTALLING

Delete the installed files from the relevant locations:

VST3:
/Library/Audio/Plug-Ins/VST3/

Audio Unit:
/Library/Audio/Plug-Ins/Components/

Standalone:
/Applications/

Removing the plugin file does not automatically remove its saved settings or
licence data.

SETTING UP LITERAL SOUNDS LABS PLUGINS IN LOGIC PRO

Logic Pro uses the Audio Unit version of a plugin. Make sure the file ending in
.component has been copied to:

/Library/Audio/Plug-Ins/Components/

Then restart Logic Pro.

EFFECT PLUGINS

Use these steps for:

- Earth Magnetic Field FX
- HeisenKaon
- ImageSonicFlux
- Phi Delay Network
- Morse Code Rhythmizer

1. Open Logic Pro and create or open a project.
2. Create an Audio Track, Software Instrument Track or Auxiliary Channel
   containing the sound you want to process.
3. In the channel strip, click an empty Audio FX slot.
4. Choose:

   Audio Units > Literal Sounds Labs > <Plugin Name>

5. Choose the Stereo version unless you are deliberately working in a surround
   or multi-output project.
6. Start playback and adjust the plugin controls.

Morse Code Rhythmizer is an audio effect. It does not generate sound by itself.
Place it on a track that already contains audio or receives a live input.

INSTRUMENT PLUGINS

Use these steps for Phi Tree Möbius Synth and any other Literal Sounds Labs
synthesizer:

1. Create a new Software Instrument Track.
2. Click the Instrument slot in the channel strip.
3. Choose:

   AU Instruments > Literal Sounds Labs > <Plugin Name> > Stereo

4. Arm the track or select it.
5. Play using a MIDI keyboard, Logic's Musical Typing keyboard, or MIDI regions.

To open Musical Typing, choose:

Window > Show Musical Typing

or press:

Command-K

IF THE PLUGIN DOES NOT APPEAR IN LOGIC

1. Close and reopen Logic Pro after installing the .component file.
2. Open Plug-in Manager.

Depending on the Logic version, use either:

Logic Pro > Settings > Plug-in Manager

or:

Logic Pro > Preferences > Plug-in Manager

3. Search for the plugin name or Literal Sounds Labs.
4. Select the plugin and choose Reset & Rescan Selection.
5. Confirm that the plugin shows as successfully validated.
6. Restart Logic after the rescan.

If validation fails:

- Confirm the .component file is in:
  /Library/Audio/Plug-Ins/Components/
- Approve the plugin in macOS Privacy & Security if macOS blocked it.
- Restart the Mac and rescan again.
- Make sure you installed the correct macOS build for your computer.

USING IMAGESONICFLUX IN STEREO

1. Create or select a stereo audio track.
2. Insert ImageSonicFlux in an Audio FX slot.
3. Choose the Stereo version.
4. Load an image or draw a path.
5. Start playback so the incoming audio follows the visual trajectory.
6. Use the plugin's stereo or binaural mode for normal two-speaker or headphone
   playback.

USING IMAGESONICFLUX IN 5.1 OR MULTI-OUTPUT

Use this only when the installed ImageSonicFlux build includes a multi-channel
Audio Unit configuration.

1. Open Logic Pro's project settings and set the project surround format to 5.1.
2. Create a surround audio track, surround bus or surround auxiliary channel.
3. Insert ImageSonicFlux in an Audio FX slot.
4. In the Audio Unit format menu, choose the 5.1 or multi-output version rather
   than Stereo.
5. Confirm your audio interface and speaker routing in:
   Logic Pro > Settings > Audio
6. Confirm the expected 5.1 speaker order in your project and audio interface.
7. Use Logic's surround meter or surround panner to verify that all channels are
   receiving signal.

A typical 5.1 channel layout is:

1. Front Left
2. Front Right
3. Centre
4. LFE
5. Surround Left
6. Surround Right

Your audio interface routing must match the speaker assignments used by Logic.

AUTOMATION

1. Press A to show automation.
2. Choose the plugin and parameter from the automation menu on the track.
3. Draw automation or move the control while using Touch, Latch or Write mode.
4. Return the automation mode to Read when finished.

SAVING PRESETS

Use Logic's plug-in preset menu at the top of the plug-in window to save a Logic
preset. The plugin may also include its own preset or settings system.

TRIAL AND LICENCE ACTIVATION

1. Open the plugin in Logic.
2. Start the trial if the plugin offers a trial mode.
3. To unlock a purchased copy, paste the Payhip licence key into the plugin's
   licence field.
4. Select Activate.
5. Save the project and reopen it to confirm the full version remains active.
