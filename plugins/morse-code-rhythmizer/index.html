Morse Code Rhythmizer

Transform incoming audio into Morse-code-driven rhythmic patterns without generating any sound of its own.

What makes it different

Morse Code Rhythmizer is an audio effect that processes the signal coming from your DAW. Enter text and its dots, dashes and spaces become a rhythmic pattern that reshapes drums, loops, vocals, drones or any other incoming sound. Choose a more evenly musical pulse or timing inspired by traditional Morse communication, then use the resulting pattern to create precise rhythmic movement, unusual phrasing and text-derived variations. Unlike Morse Code Sonifier, Rhythmizer contains no oscillators and does not operate as a synthesizer.

Perfect for

  • Rhythmic gating
  • Text-derived patterns
  • Loop transformation
  • Experimental rhythm design

Key features

  • Processes incoming audio rather than generating sound
  • Text-to-Morse rhythmic pattern conversion
  • Musical equal-beat and traditional Morse timing modes
  • Creates text-derived rhythmic movement from any audio source
  • Useful with drums, loops, vocals and sustained material
  • No oscillators or internal synthesizer voices
  • Completely free to download
