What makes it different

Morse Code Sonifier is a synthesizer that converts written text or Morse code into sound-generating rhythmic patterns. Its three oscillator sections can be shaped independently with waveform, frequency, filtering, amplitude and filter envelopes, while the built-in drum layer adds rhythmic weight. Unlike Morse Code Rhythmizer, which only processes incoming audio, Sonifier generates its own sound and can be played as an instrument.

Perfect for

  • Text-driven composition
  • Rhythmic synthesis
  • Generative sequences
  • Experimental electronic music

Key features

  • Three independent oscillator sections
  • Text and Morse-code pattern generation
  • Dedicated drum section with mute controls
  • Per-oscillator filter, amplitude envelope and filter envelope
  • Three assignable LFOs with retrigger, tempo sync and quantisation
  • Built-in reverb and delay
  • Host sync, tempo control and pattern speed multipliers
  • Play, stop, loop and recording controls
  • macOS and Windows downloads

From text to instrument

Type a word, phrase or Morse-code sequence and use it as the structural source for the performance. Dots, dashes and spaces become a repeatable musical framework that can drive oscillator voices and drums, while envelopes, filters, LFO routing and effects turn the pattern into anything from tight pulses to evolving ambient sequences.
