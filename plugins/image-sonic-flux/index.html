What makes it different

ImageSonicFlux treats an image or drawn contour as a path through sound space. The moving point can drive stereo, surround or multi-output trajectories, allowing visual geometry to become a performable spatial composition tool.

Perfect for

  • Spatial composition
  • Installations
  • Audiovisual work
  • Surround movement

Key features

  • Image and contour path mapping
  • Stereo, surround and multi-output modes
  • Drawable motion paths
  • Manual speaker routing in standalone mode
  • Path speed and direction controls
  • Designed for spatial sound work
  • Downloads will be released after testing is complete

Creative use

Transform image contours and hand-drawn shapes into expressive stereo or multi-output sound trajectories. Use it in DAWs, live setups, installations, experimental composition, ambient music, electronic production and sound-design workflows.
